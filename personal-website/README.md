# personal-website

Personal website and blog, currently hosted at
[zach.lysobey.com](https://zach.lysobey.com).

## Status

**Pre-migration.** This project is being migrated from its
[original repository](https://github.com/zachlysobey/personal-site) into
z-megarepo. See the [migration plan](./PLAN.md) for details and progress.

## Tech stack

- [Gatsby](https://www.gatsbyjs.com/) 2 (static site generator)
- React 16
- Markdown content via `gatsby-transformer-remark`
- Deployed to GitHub Pages

## Migration plan

The migration is tracked in [PLAN.md](./PLAN.md) and broken into
incremental phases:

1. Import repository with full git history
2. Align with z-megarepo conventions
3. Add GitHub Actions CI
4. Update monorepo documentation
5. Modernization (Node/Gatsby upgrades, potential Next.js migration)
